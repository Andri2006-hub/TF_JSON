## 📌 Descrição do Projeto

Este projeto tem como objetivo explorar e comparar o uso de bancos de dados relacionais e não relacionais no tratamento de estruturas JSON profundamente aninhadas.

### 🚀 Missões do Projeto

* **Missão 1:** Subir os serviços necessários utilizando **Docker Compose**, garantindo um ambiente padronizado e reproduzível com PostgreSQL e MongoDB.
* **Missão 2:** Inserir um documento JSON com **15 níveis de profundidade** em ambos os bancos:

  * No **PostgreSQL**, utilizando o tipo `JSONB`
  * No **MongoDB**, utilizando sua estrutura nativa de documentos
* **Missão 3:** Avaliar a **complexidade e desempenho** ao realizar consultas no nível mais profundo (15º nível) em cada banco.

### 📊 Entrega

O repositório contém:

* Código-fonte para configuração e execução do ambiente
* Scripts de inserção de dados
* Exemplos de queries
* Relatório comparativo analisando:

  * Tempo de execução
  * Facilidade/dificuldade de consulta
  * Diferenças entre abordagem relacional vs não relacional
